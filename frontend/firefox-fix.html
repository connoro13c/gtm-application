GTM Application - Firefox Compatibility

You're seeing this page because you're using Firefox

The application is having some routing issues with Firefox. Here are two solutions:

Option 1: Use our Express Server

Run the application with our custom Express server:

  1. Open your terminal
  2. Navigate to the project root
  3. Run: npm install (if you haven't already)
  4. Run: npm run start
  5. This will start a server that works with Firefox

Option 2: Use Chrome/Edge

The application works perfectly in Chrome-based browsers.

Technical Details

The issue occurs because Firefox handles client-side routing differently than Chrome in development mode. Our Express server fixes this by properly handling all routes.
